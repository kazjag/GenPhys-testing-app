# GenPhys Testing App
Автор: Владимир Михайлович Локонов, 1 курс магистратуры ФЭФМ МФТИ, гр. М04-506В
Описание: Веб-приложение для тестирования по оптике и фотонике. Сделано в качестве проекта для https://lms.mipt.ru/digital_department.
Предназначено для студентов кафедры фотоники ФЭФМ для проверки/самопроверки знаний. Позволяет пройти тестирование из 5 задач на различные тематики в рамках оптики и фотоники с автоматической проверкой ответов.
Инструкция по запуску локально:
1) Открыть среду разработки (PyCharm/VS-code);
2) Скопировать папку с проектом к себе на компьютер
3) Переключиться на виртуальное окружение, существующее в папке GenPhys-testing-app/.venv
4) В shell перейти в папку GenPhys-testing-app/genphysapp
5) Прописать в shell python manage.py runserver
6) В браузере перейти по http://127.0.0.1:8000
7) Готово! Можно начинать работу
